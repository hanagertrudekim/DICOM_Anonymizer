## Install

Clone the repo and install dependencies:

```bash
git clone --depth 1 --branch main https://github.com/hanagertrudekim/pyQt5-dicom-app.git your-project-name
cd your-project-name
pip install or pip3 install
```

## start

```bash
python app.py
//or
python3 app.py
```
